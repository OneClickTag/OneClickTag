# OneClickTag Project Requirements

## Tech Stack
- Backend: NestJS with TypeScript
- Database: PostgreSQL with Prisma ORM  
- Frontend: Vite + React + TypeScript + Shadcn UI
- Auth: Firebase Auth with Google SSO
- APIs: Google Tag Manager SDK, Google Ads SDK
- Deployment: Vercel

## Core Features
1. Multi-tenant SaaS architecture
2. Google OAuth integration for customer properties
3. Automated GTM tag/trigger/variable creation
4. Customer management with paginated grids
5. Campaign tracking management
6. Analytics dashboards
